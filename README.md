
# Life Copilot

[![CI](https://github.com/ryuujo1573/life-copilot/actions/workflows/ci.yml/badge.svg)](https://github.com/ryuujo1573/life-copilot/actions/workflows/ci.yml)

Life Copilot is a modular personal productivity application built with Tauri, React, and Rust. It provides a plugin-based architecture to help you manage your daily routines, focus sessions, and brain dumps in one cohesive desktop environment.

## 🚀 Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js](https://nodejs.org/) (or [Bun](https://bun.sh/))
- System dependencies for [Tauri](https://tauri.app/v1/guides/getting-started/prerequisites)

### Development

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ryuujo1573/life-copilot.git
    cd life-copilot
    ```

2.  **Install dependencies:**
    ```bash
    bun install
    ```

3.  **Start the application in development mode:**
    ```bash
    bun start
    ```

### Website & Documentation

The project includes a Qwik-based website located in the `website/` directory.

```bash
bun run doc
```

## 🧩 Features (Plugins)

- **Brain Dump**: Quick capture for your thoughts and ideas.
- **Focus Session**: Pomodoro-style timer and focus tracking.
- **Routines**: Manage and track your daily habits.
- **Todo Board**: Visual task management.
